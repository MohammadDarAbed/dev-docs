
| **==Feature==**  | ==Authentication==                                    | ==Authorization==           |
| ---------------- | ----------------------------------------------------- | --------------------------- |
| **What it does** | Verifies identity (Who is the user?)                  | Controls access/permissions |
| **Occurs when?** | First (before the authorization)                      | After authentication        |
| **Based on?**    | Credentials (the data of user like username&password) | Policies and rules          |
| **Example**      | Logging in                                            | Accessing admin dashboard   |
