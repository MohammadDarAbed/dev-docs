# 📘 Repository Pattern Documentation in .NET Core

## With Support for Soft Delete, Tracking, and Filtering

### ✅ Goal

Create a flexible, reusable, testable data access layer using the Repository pattern that:

- Encapsulates common data access logic    
- Supports:
    
    - **Soft Deletion** (`ISoftDeletable`)        
    - **Created/Updated Timestamps** (`ITrackable`)        
    - **Active Filtering** (`IFilterable`)        
- Keeps business logic clean and easy to maintain    

---

## 📐 Project Architecture

Controller → Manager → IRepo\<T> → DbContext

You’ll inject `IRepo<T>` into your **manager layer**, not `DbContext`, to keep the data access abstracted and clean.

## 🧩 Step 1: Define Base Interfaces

These interfaces will describe common behaviors across your entities.

```
public interface IEntity
{
    int Id { get; set; }
}

public interface ISoftDeletable
{
    bool IsDeleted { get; set; }
}

public interface ITrackable
{
    DateTime CreatedAt { get; set; }
    DateTime? UpdatedAt { get; set; }
}

public interface IFilterable
{
    bool IsActive { get; set; }
}
```
Use them in any entity that needs that behavior:
```
public class CustomerEntity : IEntity, ISoftDeletable, ITrackable, IFilterable
{
    public int Id { get; set; }
    public string Name { get; set; }

    // Soft Delete
    public bool IsDeleted { get; set; }

    // Tracking
    public DateTime CreatedAt { get; set; }
    public DateTime? UpdatedAt { get; set; }

    // Filtering
    public bool IsActive { get; set; }
}
```

## 🧱 Step 2: Define the `IRepo<T>` Interface

```
public interface IRepo<T> where T : class, IEntity
{
    Task<T> GetByIdAsync(int id);
    Task<List<T>> GetAllAsync();
    Task AddAsync(T entity);
    void Update(T entity);
    void Delete(T entity);
    Task SaveChangesAsync();
}
```

## 🏗 Step 3: Implement the Generic `Repo<T>`
```
public class Repo<T> : IRepo<T> where T : class, IEntity
{
    private readonly DbContext _context;
    private readonly DbSet<T> _dbSet;

    public Repo(DbContext context)
    {
        _context = context;
        _dbSet = context.Set<T>();
    }

    // Builds the query with optional filters
    private IQueryable<T> Query()
    {
        var query = _dbSet.AsQueryable();

        if (typeof(ISoftDeletable).IsAssignableFrom(typeof(T)))
        {
            query = query.Where(e => !EF.Property<bool>(e, nameof(ISoftDeletable.IsDeleted)));
        }

        if (typeof(IFilterable).IsAssignableFrom(typeof(T)))
        {
            query = query.Where(e => EF.Property<bool>(e, nameof(IFilterable.IsActive)));
        }

        return query;
    }

    public async Task<T> GetByIdAsync(int id)
    {
        return await Query().FirstOrDefaultAsync(e => e.Id == id);
    }

    public async Task<List<T>> GetAllAsync()
    {
        return await Query().ToListAsync();
    }

    public async Task AddAsync(T entity)
    {
        if (entity is ITrackable trackable)
        {
            trackable.CreatedAt = DateTime.UtcNow;
        }

        await _dbSet.AddAsync(entity);
    }

    public void Update(T entity)
    {
        if (entity is ITrackable trackable)
        {
            trackable.UpdatedAt = DateTime.UtcNow;
        }

        _dbSet.Update(entity);
    }

    public void Delete(T entity)
    {
        if (entity is ISoftDeletable softDeletable)
        {
            softDeletable.IsDeleted = true;

            if (entity is ITrackable trackable)
            {
                trackable.UpdatedAt = DateTime.UtcNow;
            }

            _dbSet.Update(entity); // soft delete
        }
        else
        {
            _dbSet.Remove(entity); // hard delete
        }
    }

    public async Task SaveChangesAsync()
    {
        await _context.SaveChangesAsync();
    }
}
```

## 🧷 Step 4: Register the Repository in DI Container

Add this to `Program.cs` or `Startup.cs`:

```
services.AddScoped(typeof(IRepo<>), typeof(Repo<>));
```

## 🧠 Step 5: Use the Repo in Your Manager.
```
public class CustomerManager
{
    private readonly IRepo<CustomerEntity> _repo;

    public CustomerManager(IRepo<CustomerEntity> repo)
    {
        _repo = repo;
    }

    public async Task AddAsync(CustomerEntity customer)
    {
        await _repo.AddAsync(customer);
        await _repo.SaveChangesAsync();
    }

    public async Task<List<CustomerEntity>> GetAllAsync()
    {
        return await _repo.GetAllAsync();
    }

    public async Task DeleteAsync(int id)
    {
        var customer = await _repo.GetByIdAsync(id);
        if (customer != null)
        {
            _repo.Delete(customer);
            await _repo.SaveChangesAsync();
        }
    }
}

```
## 🧪 Step 6: Use the Manager in the Controller

```
[ApiController]
[Route("api/[controller]")]
public class CustomersController : ControllerBase
{
    private readonly CustomerManager _manager;

    public CustomersController(CustomerManager manager)
    {
        _manager = manager;
    }

    [HttpPost]
    public async Task<IActionResult> Add(CustomerEntity customer)
    {
        await _manager.AddAsync(customer);
        return Ok();
    }

    [HttpGet]
    public async Task<IActionResult> Get()
    {
        var customers = await _manager.GetAllAsync();
        return Ok(customers);
    }

    [HttpDelete("{id}")]
    public async Task<IActionResult> Delete(int id)
    {
        await _manager.DeleteAsync(id);
        return Ok();
    }
}

```
