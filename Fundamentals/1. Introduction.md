# What is the ASP.NET Core:

ASP.NET Core is a free, open-source, and cross-platform framework developed by Microsoft for building modern web applications, APIs, and microservices. Unlike the traditional ASP.NET, it is designed to work on Windows, Linux, and macOS. It offers high performance, built-in dependency injection, and supports modern development practices like asynchronous programming, minimal APIs, and cloud deployment. It’s part of the .NET platform and allows developers to build scalable and secure applications.

 هو إطار عمل مجاني ومفتوح المصدر ومتعدد المنصات تم تطويره من قبل شركة مايكروسوفت لبناء تطبيقات الويب الحديثة وواجهات برمجة التطبيقات (APIs) والخدمات المصغّرة (Microservices). بخلاف ASP.NET التقليدي، يمكن تشغيله على أنظمة ويندوز ولينكس وماك. يتميز بأداء عالٍ، ودعم مدمج لحقن التبعيات (Dependency Injection)، ويدعم أساليب التطوير الحديثة مثل البرمجة غير المتزامنة (Asynchronous) وMinimal APIs والنشر السحابي. وهو جزء من منصة .NET ويتيح للمطورين بناء تطبيقات آمنة وقابلة للتوسع.

# ASP.NET VS ASP.NET Core:
| Feature                  | **ASP.NET (Framework)**               | **ASP.NET Core**                                         |
| ------------------------ | ------------------------------------- | -------------------------------------------------------- |
| **Release Year**         | 2002                                  | 2016                                                     |
| ==**Platform**==         | ==Windows-only==                      | ==Cross-platform (Windows, Linux, macOS)==               |
| **Performance**          | Slower compared to Core               | High performance, lightweight, faster                    |
| **Modularity**           | Monolithic (System.Web)               | Highly modular via NuGet packages                        |
| **Hosting**              | IIS only                              | Self-hosting, Kestrel, IIS, or any server                |
| **Dependency Injection** | Limited, third-party only             | Built-in and extensive support                           |
| **Configuration**        | Web.config (XML-based)                | appsettings.json and environment-based configuration     |
| **Development Model**    | Tightly coupled with System.Web       | Modern, decoupled, supports minimal APIs and Razor Pages |
| **Open Source**          | Partially open                        | Fully open-source and community-driven                   |
| **Latest Features**      | No longer actively updated            | Actively developed with .NET updates                     |
| **Use Case Today**       | Legacy applications, maintenance only | New development, modern applications                     |

## What is the HTTP?

HTTP (Hypertext Transfer Protocol) is the main protocol used for sending and receiving information on the internet, especially between web browsers and servers.
